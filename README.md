# Substrate Validator Nodes IaaC
This repo is designed to automate Substrate Validator nodes deployment and configuration. The repository designed to support
multiple clouds including AWS, Azure and GCP.

![](https://i.imgur.com/4pFxGWI.png)

## Contents
The IaaC code in this repo has two main parts:
- Infrastructure Provisioning using Terraform
- Configuration using Ansible

If you have an established infrastructure, you need only need the configuration part which will be described under ansible folder.
