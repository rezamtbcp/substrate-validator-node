## Setup and Configure Environment Tooling
We are going to use Ansible as our configuration management tool and setting up every nodes that has been deployed.

There are plenty online guidelines and tutorials to show you how to install Ansible on your machine. Once its installed, 
run the following commands to make sure your environment is working. 

```bash
ansible --version

python --version

```
